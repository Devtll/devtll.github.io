为什么写博客

早在2015年初我就搭建了这个博客，当时也毅然决然的要坚持写下去，但是到今天 (2016-04-13) 已经过去一年了，我的博客里依旧空空如也，一篇文章也没有写。OH MY GOD, 不经要问这一年我都在忙些什么？我试着去总结了下：

  • 懒惰 ( 这是主要原因 )
  • 无从下笔 ( 简单的不愿写，复杂的不会写 )
  • 文笔不好 ( 理科生永远的痛 )
  • 工作忙 ( 这种情况可以忽略不计，时间就像乳沟，挤挤总会有的 )

那我为什么又要开始写博客了呢？我也试着总结了写作的好处：

  • 提高自己对知识的掌握层次
  • 提高表达和沟通的能力
  • 记录你的学习历程
  • 结识志同道合的新朋友
  • 面试加分
  • …

项目开发中也许学到了一个技能，或者一个知识点，但是通过写博客会加深巩固自己学习的东西，自己写不出来可能说明你对这个知识点理解还不够深入。

程序员大都不善于沟通，因为日常的工作大部分时间都是面对电脑。许多人周末也喜欢当一个技术宅，待在家里上网、看电影或者玩游戏来消遣。长时间的面对机器，使得我们的语言表达能力极度衰退。但是通过坚持写博客，自己的表达能力与表达逻辑会慢慢锻炼出来。

如何坚持写下去

首先我们并不是作家，所以，你不需要一定擅长写作，只需做到清楚的表达你的观点即可。刚开始我们可以尝试写一些不是那么深入的文章，也不必太过追求完美。写作能力的提高本来就循序渐进的，刚开始写简单一些，总归比不写要好得多，这样也会渐渐培养起我们写作的习惯。

总结

总之，我们应该博客，并且要坚持写下去。通过写博客提升自己能力，也许有一天你会突然发现，原来自己已经走了这么远！
